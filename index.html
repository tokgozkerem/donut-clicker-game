About

Hi there! Donut Clicker is lovingly crafted by ktz and doa.

Here, you'll find the latest updates and news about the game.

Got a question, found a bug, or just want to say hi? Feel free to reach out at ktz@donutclicker.net. We'd love to hear from you!

Updates

Big Update: Version 1.3.1

  • Auto-buyer added, so you won't miss progress even when you're away from the game.
  • New upgrades introduced to enhance your production.
  • Baker's Recipe Book in the Baker panel is now active, currently providing boosts for 2 buildings.
  • Quest reward multipliers will no longer stack.
  • Issues with the quest screen have been resolved.
  • UI improvements for a more seamless experience.
  • Optimizations implemented, and several bugs have been fixed.
  • Fixed an issue where images were repeatedly re-downloaded.

Big Update: Version 1.3.0

  • New quests added with exciting challenges to complete!
  • Quest rewards have been adjusted and balanced.
  • UI improvements for a better mobile experience.
  • New notification system added for key game events!
  • Baker panel introduced (still in development, more features coming soon!).
  • Building contributions to total production are now calculated and displayed.
  • Prestige now grants 5 cursors for free at the start!
  • UI fixes applied to multiplier rewards from quests.
  • Several bugs have been resolved for smoother gameplay.
  • Cursors around the donut have been removed for a cleaner design.

Small Update: Version 1.2.9

  • Added Quest menu with a variety of new missions!
  • New decorations drop when clicking the donut.
  • Performance enhancements.
  • Snow animation upgraded for a smoother experience.
  • UI improvements and bug fixes.

Big Update: Version 1.2.8 🎄

  • ❄️ A temporary Christmas theme has been added!
  • ☃️ Snowfall now increases based on your production rate. You can enable or disable it in the Settings menu.
  • 🎨 Store item images have been updated with a festive winter theme.
  • 🔄 Quest, combo, and prestige icons have also been redesigned to match the Christmas spirit.
  • 🍩 The big donut image has been replaced with a special holiday version.
  • 🎁 A Christmas Box has been added, offering surprise rewards every 10 to 15 minutes. Be quick and attentive to catch it!
  • 📐 UI improvements have been made for a better experience.
  • 🐞 Several bugs have been fixed to enhance stability.

Enjoy the holiday season and happy clicking! 🎄🍩

Small Update: Version 1.2.7

  • Cursor upgrade images replaced with new ones!
  • Cursors now orbit around the donut!
  • Upgrade prices have been rebalanced!
  • Ore production reduced to 3 for a more balanced experience!
  • Bakery name can now be customized!
  • Sound on/off toggle added!
  • Manual save option now available—save your progress anytime!

Big Update: Version 1.2.6

  • New click-based upgrades added.
  • New mine panel added after purchasing a mine (open by clicking on the mine image).
  • New worker & ore system introduced.
  • Market specifically for ores added, refreshing regularly (more to come!).
  • Performance improvements made.
  • Various bug and UI issues fixed.
  • Changes to prestige points implemented.

Big Update: Version 1.2.5

  • Two new buildings added
  • New upgrades introduced
  • Stats-info panel implemented
  • UI improvements and bug fixes
  • Pricing adjustments made
  • Portal image updated
  • Complete overhaul of script.js
